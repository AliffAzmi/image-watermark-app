# image-watermark-app
Simple Node.JS, ImageMagick app to add watermark in image.

**Installation**

1. Install depend.
```
npm install 
```

2. Requires imagemagick CLI tools to be installed. See (https://imagemagick.org/script/download.php).

3. Run
```
npm run dev 
```

4. Visit `http://localhost:4002`
